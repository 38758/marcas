En este manual hablaré sobre algunos de los elementos básicos que contiene un documento html, comenzando con algunos elementos básicos como los headings, los paragraphs, o los links, y definiré brevemente otros elementos que componen el documento. El documento html, para ser un documento html, debe contener obligatoriamente los siguientes elementos:
Nota: Todos los elementos comienzan con<...> y se cerrarán con </...>

  • Elemento doctype.
    Este elemento <doctype> es especialmente importante, por que define que el documento esta bajo el estandar de HTML 5.
  • Elemento html
    Este elemento representa la raíz de un documento <html> o XHTML. Todos los demás elementos deben ser descendientes de este elemento.
  • Elemento head
    Representa una colección de metadatos acerca del documento, incluyendo enlaces a, o definiciones de, scripts y hojas de estilo.
  • Elemento body
    Representa el contenido de un documento HTML. Solo puede haber un elemento body en un documento.

    • Aquí es donde cualquiera puede empezar a dar sus primeros pasitos...aunque si te parece, puedes seguir probando tú mismo en la página de w3schools.AQUI

    • Elemento title
      Define el título del documento, el cual se muestra en la barra de título del navegador o en las pestañas de página. Solamente puede contener texto o números y cualquier otra etiqueta contenida no será interpretada.
    • Elemento meta
      Define los metadatos que no pueden ser definidos usando otro elemento HTML.
    • Elemento br
      Representa un cambio temático entre elementos a nivel de párrafo, por ejemplo: un cambio de escena en una historia o un cambio de tema dentro de una sección.
    • Elemento span
      Es un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea.
    • Elemento p
      Si queremos especificar que se trata de un párrafo, podríamos encerrar el texto con la etiqueta de párrafo <p>
    • Elemento a
      El elemento a crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL.
    • Elemento img
      Define una imagen en una página HTMLLas imágenes no se insertan técnicamente en una página HTML, las imágenes están vinculadas a páginas HTML. La etiqueta <img> crea un espacio de retención para la imagen referenciada..

  1. Elemento form
    El elemento <form> se usa para crear un formulario HTML para la entrada del usuario. El HTML de un formulario web está compuesto por uno o más controles de formulario, además de algunos elementos adicionales que ayudan a estructurar el formulario general; a menudo se los conoce como formularios HTML. Los controles pueden ser campos de texto de una o varias líneas, cajas desplegables, botones, casillas de verificación o botones de opción, y se crean principalmente con el elemento <input>.
  2. Elemento input
    Se usa para crear controles interactivos para formularios basados en la web, que reciban datos del usuario. La forma en que <input> funciona varía considerablemente dependiendo del valor de su atributo <type>.
  3. Elemento label
    El elemento <label> representa una etiqueta para un elemento en una interfaz de usuario. Este puede estar asociado con un control mediante la utilizacion ubicando el control dentro del elemento <label>.
  4. Elemento select
    Representa un control que muestra un menú de opciones. Las opciones contenidas en el menú son representadas por elementos <option>.
  5. Elemento option
    La etiqueta <option> define una opción en una lista de selección o <datalist> y mostrando su contenido mediante su atributo <value>.
  6. Elemento textarea
    La etiqueta <textarea> define un control de entrada de texto de varias líneas mediante sus atributos <rows> y <col>.
  7. Elemento input
    El elemento <input> se usa para crear controles interactivos para formularios basados en la web, que reciban datos del usuario. La forma en que <input> funciona varía considerablemente dependiendo del valor de su atributo <type>.
    • Type
      El tipo de control a mostrar. Su valor predeterminado es "text", si no se especifica este atributo. Los posibles valores son: button, checkbox, color, date, datetimelocal, email, file, hidden, image, month, number, password, radio, range, reset, search, submit, tel, text, time, url,week,
  8. Elemento Iframe
    Representa un contexto de navegación anidado, el cual permite incrustrar otra página HTML en la página actual.
  9. Elemento audio
    Especifica una forma estándar de incrustar audio en una página weba, y para poder usar los elementos de control, deberemos utilizar el atributo controls.
  10. Elemento video
    Se utiliza para incrustar vídeos en un documento HTML o XHTML.
